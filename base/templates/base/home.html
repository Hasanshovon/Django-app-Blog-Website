projects

  • Tampere City Bus and Tram Audio Source Classification with Librosa and Machine Learning

    In this project, we developed a machine learning model to classify audio recordings of Tampere City buses and trams. Using the Librosa library, we extracted key audio features like MFCCs, Mel spectrograms, and Chroma. To combat the high dimensionality of audio data, we implemented Principal Component Analysis (PCA) for feature reduction. The model combined LSTM networks and supervised classifiers to accurately distinguish between bus and tram sounds. This work demonstrates the potential of advanced audio processing and machine learning in urban sound analysis.

  • Satellite Image Classification Using TensorFlow

    In this project, we utilized the RSI-CB256 dataset, comprising 4 mixed classes from sensors and Google Maps snapshots, to advance remote sensing (RS) image interpretation for AI and computer vision applications. We employed a Convolutional Neural Network (CNN) with a 64x64x3 input layer, integrating Conv2D layers, Batch Normalization, MaxPool2D, Dropout, Flatten, Dense layers, and Softmax activation. Our CNN model achieved a notable test accuracy of 94.23%. For data preparation, TensorFlow's Image Data Generator was used for data augmentation, incorporating techniques such as rescaling, rotation, zooming, and horizontal flipping, enhancing the model's robustness and generalization capabilities.

  • Credit Card Fraud Detection Using GAN and Traditional Algorithms

    This project entailed analyzing a dataset of 284,807 European cardholder transactions from September 2013, including 0.172% fraudulent cases, to detect fraud. Features were mostly PCA-transformed, except for 'Time' and 'Amount'. The focus was on the 'Class' feature for fraud identification. Given the class imbalance, the Area Under the Precision-Recall Curve (AUPRC) was the key metric. We applied Isolation Forest, Local Outlier Factor (LOF), and One Class SVM, with Isolation Forest achieving the best results at 27% precision. Additionally, a GAN was used for synthetic data generation, reaching a high discriminator accuracy of 99.22% but a low F1 score in fraud detection. This project highlights the challenges and complexities of fraud detection in imbalanced datasets.
